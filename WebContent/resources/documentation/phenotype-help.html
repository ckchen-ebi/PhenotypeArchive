The gene associations table on the phenotype page comes in two slightly different flavours:
- phenotype terms that are directly annotated to data
- phenotype terms whose gene associations have been inferred. Thus, for a top level term, in the 5th column we have a phenotype term indicating the MP term directly associated to the respectivegene (allele??).

We offer 2 export options for the data in the table : in a text file with tab separated variables () and as an excell sheet (). In the table on our page entries lines are collapsed if all fields are identical except the gender. In the export file, for a html table line representing both genders we will export 2 lines in both XLS and TSV files.
