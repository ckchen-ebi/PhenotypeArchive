Explore the diverse entry points to mouse phenotype data.

Our phenotype page currently consists of two parts: the phenotype information/details and the table of gene variants associated with the current phenotype.

Phenotype details

The synonyms and definition we provide come from the mouse anatomy ontology (MP). This is also why we provide the MP id of the current phenotype page as well as the link to it on the MGI MP browser.

The procedures come from legacy data pipelines such as EUMODIC, GMC.

Table of associations

Rows are collapsed based on sex. For more readability we group entries where all fields are identical except the sex to one row. These rows are identified by a both-sexes icon (). The total number of results shown on top of the table will double count this rows (once for females and once for males).

Direct vs. inferred associations

The gene associations table on the phenotype page comes in two slightly different flavors:

  • phenotype terms that are directly annotated to data
  • phenotype terms whose gene associations have been inferred. Thus, for a top level term, in the 5th column we have a phenotype term indicating the MP term directly associated to the respective gene (allele??).

Downloading results

We offer 2 export options for the data in the table:

  • TSV, text file with tab separated variables ()
  • ELS, Microsoft Excel spread sheet ()

Please note that in the table displayed on our page entry lines are collapsed based on sex. That is, if for 2 lines all fields are identical except the gender, they will be shown together for a better user experience. In the export file however we export all lines separately, to allow easier further processing of the data. This holds for both XLS and TSV files.
