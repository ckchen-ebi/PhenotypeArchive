                  EMBL-EBI Phenotype Archive Release Notes

Introduction

The EMBL-EBI Phenotype Archive is an open source project to store, analyse and present genotype associated phenotypes information on the web from knock-out mice. This project is funded by the NIH KOMP2 grant and supported by IMPC consortium.

See https://github.com/mpi2/PhenotypeArchive/wiki for more information.
